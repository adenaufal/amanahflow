## CARA PAKAI DI WEB UI:

1. Buka https://elevenlabs.io/speech-synthesis
2. Pilih voice: U3dExJoUNcmTY5H6GMuG (atau voice library lain yang kamu suka)
3. Settings:
   - Stability: 0.70
   - Clarity + Similarity: 0.80
   - Style: 0 (Natural)
   - Speed: 0.92x (jika ada slider speed)
4. Copy-paste narasi per segmen (tanpa header "SEGMENT X")
5. Generate → Download MP3
6. Rename file sesuai output name (seg1-hook.mp3, dst)
7. Taruh di I:/Github/hackathon/amanahflow/video/public/audio/

## TIPS:

- **Pause otomatis:** Web UI biasanya handle pause dari paragraph breaks (blank lines) dengan lebih baik
- **Jika pause kurang:** Tambah blank line ekstra antar kalimat
- **Jika pause terlalu panjang:** Gabungkan kalimat jadi satu paragraph
- **Preview dulu:** Dengarkan hasil sebelum download, bisa regenerate jika perlu
- **Batch mode:** Beberapa voice di web UI ada "batch mode" untuk generate multiple scripts sekaligus

## ALTERNATIVE: Jika Voice Support SSML

Jika voice yang kamu pilih support SSML tags (cek di voice description), bisa pakai format ini untuk pause lebih presisi:

```
Hari ini, ada pengurus masjid yang masih ngetik manual ke spreadsheet. <break time="1.5s" /> Sambil balas konfirmasi transfer satu per satu di WhatsApp.
```

Tapi untuk voice library biasa, format di atas (dengan paragraph breaks) sudah cukup.
